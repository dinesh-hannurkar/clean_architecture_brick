# New Developer Onboarding Guide

Welcome to the team! 🎉 This guide will help you get setup and productive on Day 1.

## 🏁 Day 1 Checklist

- [ ] Join the Slack channel `#project-app`.
- [ ] Get access to the GitHub repository.
- [ ] Install required software (see below).
- [ ] Clone and run the project.
- [ ] Read the [Clean Architecture Guide](../architecture/clean-architecture.md).

## 💻 Environment Setup

### 1. Install Tools
1. **VS Code** (Recommended) or Android Studio.
2. **Flutter Version Management (FVM)** (Optional but recommended).

### 2. VS Code Extensions
- Flutter & Dart
- Flutter Bloc
- Coverage Gutters
- Todo Tree

## 🚀 Your First Run

1. **Clone**: `git clone ...`
2. **Dependencies**: `flutter pub get`
3. **Run**: `flutter run`

## 🧪 Your First Contribution

1. Pick a "Good First Issue" from the board.
2. Create a branch: `git checkout -b feature/xyz`.
3. Implement changes.
4. Run tests: `flutter test`.
5. Submit PR.
