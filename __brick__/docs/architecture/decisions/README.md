# Architecture Decision Records (ADRs)

This directory contains the record of significant architectural decisions made for Project App.

## What is an ADR?

An ADR is a short text file that captures a single architectural decision, such as a choice of technology, a structure, or a pattern.

## Process

1. **Copy the Template**: Use `docs/templates/adr-template.md`.
2. **Draft**: Create a new file `ADR-XXX-title.md` in this directory.
3. **Review**: Submit a PR for discussion.
4. **Merge**: Once accepted, merge the PR.
